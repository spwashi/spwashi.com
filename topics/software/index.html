## Note

Files should remain legible.

- durable

- inspectable

Spw frame

#>note
#:layer #!pragmatics

^"summary"{
 claim: "Files should remain legible."
 tags: ["durable", "portable", "inspectable"]
}

The source stays readable while becoming addressable to parsers, metadata systems, and UI surfaces.

Core forms

  • #>frame Name a frame so it can be linked, queried, and projected.
  • #:layer Mark the interpretive layer or role of the frame.
  • ^"object"{ } Hold structured content in a stable, inspectable body.
  • ~"..." Carry references, paths, or literal anchors.
  • ?[...]{...} Store questions, probes, or interactive interpretive units.

Composite example

#>frame_name
#:layer #!meaning_time

^"object"{
 key: "value"
 list: ["one", "two"]
 ref: ~"./other-surface.spw"
}

?["What changes when a note becomes queryable?"]{
 !probe{ "project this frame into outline, graph, and editable card views" }
}

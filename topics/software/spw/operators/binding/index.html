= Binding

The binding operator names a local value so it can be reused, compared, pinned, or categorized. It is the moment salience gets an address.

previous: & operator atlas next: $
^"binding_contract"{

Contract

Interaction

Name, pin, or categorize a value. Binding should be reversible or inspectable so names do not become hidden state.

Metaphysics

Binding is categorization: attention stops sliding and agrees to call something by a stable name.

Brace Physics

A binding brace stores charge as a local anchor. Too many hidden anchors create pressure, so bindings must be visible to inspection.

Symmetry

Prefix =x primes a value for naming. Postfix x= marks the value as bound after it resolves.
